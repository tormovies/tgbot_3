# Бот MT4/MT5 — индикаторы и советники

**Репозиторий:** https://github.com/tormovies/tgbot_3.git

## Запуск (PHP)

1. Установи PHP 7.4+ (с поддержкой JSON и HTTP-запросов).
2. Создай файл `.env` и укажи токен:
   ```
   BOT_TOKEN=твой_токен_от_BotFather
   ```
3. В папке проекта выполни:
   ```bash
   php bot.php
   ```

Состояние диалогов хранится в папке `data/` (создаётся автоматически).

Команды: `/start`, `/catalog`, `/prices`, `/order`, `/help`, `/cancel` (в сценарии заказа).
